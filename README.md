# Rohan Sakthivel — Portfolio (React + Vite)

Local setup

1. cd portfolio
2. npm install
3. npm run copy-resume   # copies your local resume into the app public folder
4. npm run dev

Optional

- `npm run import-site` will fetch your existing site HTML and save it to `src/data/importedSite.html` for reference.

Notes

- Make sure Node 18+ is installed for `fetch` in `import-site.js` or install `node-fetch` and adapt the script.
- Edit `src/components/Projects.jsx` to add your real projects and links.
